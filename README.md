# Creative Agency

Key features:
* Dynamic website, backed by NodeJS and MongoDB
* CRUD functionalities for add, edit, remove volunteer services
* Organized admin panel

Libraries used:
* React JS
* React Router Dom
* Firebase (for hosting and image storage)
* React Hook Form
* Bootstrap 4
